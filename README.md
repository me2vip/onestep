# 一级台阶 · 狠话与真心

一个基于 Canvas 的交互式语录展示网页，以"一级台阶"为视觉意象，承载或尖锐或温暖的话语，配合粒子特效与合成环境音，营造沉浸式体验。

> 往上一步，是狠话；往下一步，是真心。

## 功能特性

- **三种模式**：狠话（粉红霓虹）、真心（暖金柔光）、混合（电光蓝 + 粉金交替点缀）
- **Canvas 台阶动画**：带透视缩放的阶梯持续滚动，每级台阶承载一句语录
- **粒子特效**：底部上升的萤火光点，强度可调
- **背景音乐**：内置 Web Audio API 合成环境音（低频和弦 + 白噪音），也支持上传本地音乐
- **控制面板**：毛玻璃风格，可调节滚动速度、粒子强度、文字大小、音量等
- **随机语录**：一键高亮随机一条语录
- **响应式布局**：PC 端面板悬浮右侧，移动端折叠至底部

## 快速开始

无需构建工具，直接用浏览器打开即可：

1. 克隆仓库
   ```bash
   git clone <repo-url>
   cd onestep
   ```
2. 用浏览器打开 `index.html`（或使用本地服务器）
   ```bash
   # 例如使用 Python
   python -m http.server 8080
   # 然后访问 http://localhost:8080
   ```
3. 点击「开始体验」进入

## 技术栈

- **纯前端**，零依赖，无需构建
- HTML5 Canvas 2D API（核心渲染）
- CSS3（backdrop-filter、gradients、响应式）
- Web Audio API（合成环境音）
- ES6+ 原生 JavaScript

## 模块说明

### `js/data.js` — 语录数据

导出 `window.QUOTES`，包含 `harsh`（狠话）和 `true`（真心）两个数组，每条记录格式：

```js
{ type: 'harsh', text: "你若安好，那还得了。", author: "网络" }
```

### `js/canvas.js` — CanvasScene

核心渲染类，负责：

- 台阶 3D 透视绘制（近宽远窄 + 顶面/立面分层）
- 语录文字沿台阶排布，自动截断超长文本
- 粒子系统（上升光点 + 闪烁 + lighter 混合模式）
- 三套颜色方案随模式切换
- 混合模式支持奇偶交替排列

### `js/audio.js` — AudioController

音频控制类，支持双音源：

- **合成音**：三个正弦波和弦（A2/E3/A3）+ LFO 调制 + 低通滤波白噪音
- **外部音源**：用户上传本地音频文件，自动切换播放

淡入淡出控制，音量实时调节。

### `js/app.js` — 主入口

串联所有模块，绑定控制面板事件：

- 模式切换（狠话 / 真心 / 混合）
- 混合模式奇偶交替开关
- 滚动速度、粒子强度、文字大小滑块
- 音乐开关、音量、上传
- 随机语录高亮、重置滚动
- 主显语录 overlay 动画

## 控制面板参数

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| 模式 | 狠话 / 真心 / 混合 | 狠话 | 切换语录类型与配色 |
| 滚动速度 | 0.2x – 3.0x | 1.0x | 台阶滚动速率 |
| 粒子特效 | 0 – 100 | 50（中） | 粒子数量与密度 |
| 文字大小 | 12px – 28px | 16px | 台阶上语录字号 |
| 背景音乐 | 开/关 | 关 | 合成环境音或上传音乐 |
| 音量 | 0% – 100% | 60% | 音乐音量 |

## 配色方案

| 模式 | 棱边发光 | 顶面高光 | 粒子色 |
|------|----------|----------|--------|
| 狠话 | `#ff2e9c` 霓虹粉 | `#ff6fae` | 粉 / 青 |
| 真心 | `#ffd86b` 暖金 | `#ffe4a3` | 金 / 白 |
| 混合 | `#00e5ff` 电光蓝 | `#a6c7ff` | 青 / 粉 / 金 |
